# 🏦 Добро пожаловать в BEN

**BEN** (Banking Ecosystem Network) — система для автоматического выявления и анализа фрод-инцидентов в банке.

Когда клиент подаёт жалобу, BEN находит мошенника по базе данных транзакций, собирает его полный профиль
из нескольких источников (банк, мобильный оператор, маркетплейс) и отправляет сводку операторам в Telegram.

---

## 🚀 Как это работает

```
Жалоба клиента
      │
      ▼
 BEN API извлекает сумму из текста жалобы
      │
      ▼
 Поиск транзакции в ecosystem_data.db
      │
      ▼
 Сборка профиля мошенника (банк + звонки + маркетплейс)
      │
      ▼
 Уведомление операторам в Telegram
```

---

## 📦 Компоненты системы

!!! note "BEN API"
    FastAPI-бэкенд на `api.py`. Предоставляет REST-эндпоинты для работы с жалобами,
    расследованиями и профилями пользователей. Документация Swagger доступна на `/docs`.

!!! note "Fraud Investigator"
    Движок расследований на `fraud_analysis.py`. Извлекает суммы из текста жалоб регулярками,
    ищет транзакции в БД и строит полные профили с тегами риска.

!!! note "Telegram Bot"
    Бот на aiogram 3.x. Поллит новые жалобы каждые 60 секунд, рассылает уведомления операторам
    и позволяет расследовать кейсы прямо в мессенджере.

## 🗄️ Источники данных

| Источник | Таблица | Что даёт |
|---|---|---|
| Банк | `bank_clients`, `bank_transactions` | Клиенты, счета, переводы |
| Мобильный оператор | `mobile_build` | История звонков |
| Маркетплейс | `market_place_delivery`, `ecosystem_mapping` | Адреса доставок |
| Жалобы | `bank_complaints.tsv` | Тексты жалоб клиентов |
