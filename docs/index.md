# Добро пожаловать в проект BEN

**BEN** (Banking Ecosystem Network) — это система для анализа фрод-инцидентов. Когда появляется жалоба от клиента банка, наш BEN находит мошенника по базе данных клиентов банка. Он соберет полную информацию о моешннике
и предоставит ее вам, используя нашу веб оболочку. Жалобы обрабатываются автоматически
и когда обработка кончается, присылает вам в тг бот уведомление об новом мошеннике с краткой информацией о нем.

!!! note "Статус проекта"
    Проект BEN находится в стадии активной разработки. 🚀

!!! warning "Важно для разработчиков"
    Перед запуском тестов убедитесь, что вы настроили `.env` файл.
