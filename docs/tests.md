# 🧪 Тестирование

Проект покрыт **Unit-тестами на pytest**. Тесты живут в ветке `dev_sasha_test`
и запускаются автоматически при деплое документации.

---

## 📊 Покрытие

| Модуль | Тесты | Что проверяется |
|---|---|---|
| `fraud_analysis` | ✅ | Извлечение сумм, поиск транзакций, генерация тегов |
| `db_creator` | ✅ | Генерация БД, структура таблиц, целостность данных |

---

## ▶️ Запуск тестов

```bash
# Клонируй ветку dev_sasha_test
git checkout dev_sasha_test

# Установи зависимости
pip install pandas numpy faker pytest

# Запусти все тесты
pytest

# Или с подробным выводом
pytest -v
```

!!! warning "Важно"
    Перед запуском тестов убедись что настроен `.env` файл
    и сгенерирована тестовая база через `python db_creator.py`.

---
